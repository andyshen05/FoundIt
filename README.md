# Lost & Found Management System
CS3200 – Database Design – Final Project

This project implements a Lost & Found platform called FoundIt
It supports two main user roles:

- Jayson Tatum: regular user who submits lost reports, sets rewards, and receives notifications.
- Payton Pritchard: inventory staff who manages found items and storage.

Note: 2 Other profiles, Francis Smith (user who found an item) and Braden Ward (system admin), routes/pages are not included because team members dropped the class/did not contribute

## Features
- Submit and manage lost item reports
- Create, update, and remove rewards
- Add and update found inventory items
